What is it?
-----------
EasyWords is an dictionary-app. 
The app provides some exercises that can help to learn english words from russian.
It is based on PyQt5.

Installation
------------
To run the programm you need to download dictionary.db from https://disk.yandex.ru/d/QQloKLl7SO5xWg
and then run EasyWords.exe (or main.py).

Required files
--------------
STYLE.css
EasyWords.exe (or main.py + DMS.py)
dictionary.db
AppIcon.png
